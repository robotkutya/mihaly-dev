Hi, I'm Mihaly 👋

This is my digital business card & online resume. It processes LinkedIn's data archive and transforms it into an open, machine-friendly website.

Humans are expected to ingest via the UI, but machines are welcome too!

- `/profile.json`: canonical structured public profile data.
- `/profile.md`: markdown rendering of the same profile facts.
- `/AGENTS.md`: instructions for agents consuming the public profile.
- `/llms.txt`: agent-discovery index linking to the canonical resources above.
- `/robots.txt`: crawler policy that allows indexing and AI answer/search discovery while blocking separable training-only crawling.

If you're human, try out the oracle! 🖖
